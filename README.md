## 关于

专注 **嵌入式硬件** 与 **现代 AI** 的衔接：把 LLM / RAG 能力落到边缘与实时系统里，搭建数据驱动的低延迟架构。当前延伸学习 **ROS2**、**TinyML**。  
*Mindset: system architecture over code-only.*

- LLM 与嵌入式系统集成  
- RAG 与智能流水线  
- AI + IoT + 边缘计算  
- 实时与可观测架构
